Руководство пользователя

Данная программа реализует расчёт справедливой цены опциона по модели Хестона
как по историческим данным, так и непосредственно по параметрам рынка.

1) Расчет справедливой цены опциона по историческим данным или параметров
рынка из модели Хестона:

	- Нажмите кнопку "Выбрать файл"

	- Выберите .csv-файл, содержащий исторические данные в требуемом формате

	- Нажмите на кнопку "Расчитать параметры"

	По окончании расчёта параметры рынка появятся в полях "Параметры рынка".

	- Если требуется найти справедливую цену опциона, нажмите на кнопку
	"Расчитать цену"

	Справедливая стоимость опциона появится в поле "Справедливая стоимость опциона"

	Формат .csv-файла с историческими данными: в качестве разделителя между
	значениями используется “;”, в качестве десятичного разделителя - “.”,
	разделитель между строками - “\n”. Столбцы в файле должны содержать следующее
	строго в данном порядке:
		1) время до экспирации в долях года;
		2) спотовая цена опциона;
		3) цена спроса на базовый актив (bid);
		4) цена предложения на базовый актив (ask);
		5) цена исполнения (strike);
		6) значение безрисковой ставки (от 0 до 1).

	Все цены должны быть указаны в одной и той же валюте, соответственно, как
	результат, справедливая стоимость опциона будет подсчитана в той же валюте.

	Также требуется указать параметры генетического алгоритма, такие как:

	- Размер популяции. Увеличение размера популяции приводит к ускорению
	сходимости алгоритма, при этом значительно увеличивая время вычислений.
	Рекомендуемый размер — 40.

	- Количество поколений. Генетический алгоритм не имеет одного чёткого критерия
	остановки, поэтому как критерий остановки был выбран критерий числа поколений:
	при генерации заданного числа поколений алгоритм останавливается.
	Соответственно, при увеличении числа поколений, увеличивается точность.

	- Вероятность кроссовера. Вероятность кроссовера, с одной стороны, увеличивает
	сходимость к ближайшему экстремуму, с другой стороны, сильно увеличивает
	трудоёмкость алгоритма. Эмпирически были получены рекомендуемые значения для
	вероятности кроссовера в 0.75-0.85.

	- Вероятность мутации. Мутация — внесение случайных изменений в некоторые
	элементы популяции — способствует избежанию попадания в локальные экстремумы и
	игнорированию глобального. Рекомендуемое значение — 0.1.

2) В случае известных параметров рынка просто введите параметры в
соответствующие поля и нажмите кнопку "Расчитать цену".

Параметры рынка, подбираемые генетическим алгоритмом:

	- корреляцию двух винеровских процессов (rho);
	- скорость возврата (kappa);
	- уровень возвратности (theta);
	- среднеквадратичное отклонение волатильности (sigma);
	- текущую волатильность (v0);
Текущие известные параметры рынка:

	- время до экспирации в долях года (от 0 до 3);
	- спотовая цена опциона (от 0 до 100000);
	- цена спроса на базовый актив (bid) (от 0 до 100000);
	- цена предложения на базовый актив (ask) от (0 до 100000, ask > bid);
	- цена исполнения (strike) (от 0 до 100000);
	- значение безрисковой ставки (от 0 до 1);

Также для расчёта стоимости опциона по формуле Хестона необходимо указать
количество точек, по которым производится расчёт суммы ряда Фурье для ускорения
интегрирования. Это количество является степенью числа 2, поэтому в
соответствующее поле требуется ввести показатель степени. При увеличении числа
точек увеличивается точность, но также увеличивается и время вычислений.
Оптимальное значение показателя степени — около 12-14.
